§Caso de Uso: Detener Terapia

Este caso de uso se encarga de detener la terapia visual, ocultando el overlay y liberando sus recursos.

§Flujo

  1. Verifica que el overlay esté activo.
  2. Si está activo, llama al puerto OverlayPort::hide().

§Ejemplos

use terapia_visual_domain::use_cases::stop_overlay_therapy;
use terapia_visual_domain::ports::OverlayPort;

stop_overlay_therapy::stop_overlay_therapy(overlay).await?;
