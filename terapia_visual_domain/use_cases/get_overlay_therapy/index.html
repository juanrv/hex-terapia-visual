§Caso de Uso: Obtener Configuración de Terapia

Este caso de uso obtiene la configuración actual de la terapia desde el almacenamiento.

§Comportamiento

  • Si existe una configuración guardada, la devuelve.
  • Si no existe o hay un error, devuelve la configuración por defecto.

§Ejemplos

use terapia_visual_domain::use_cases::get_overlay_therapy;
use terapia_visual_domain::ports::ConfigStorage;
use terapia_visual_domain::domain::OverlayTherapyConfig;

let config = get_overlay_therapy::get_overlay_therapy(storage).await;
