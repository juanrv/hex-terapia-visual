§Módulo de Layouts

Define los patrones de división de la pantalla para la terapia visual.

Cada variante de Layout determina cómo se distribuyen las zonas de color sobre la pantalla del usuario. Actualmente se soportan:

  • Vertical: dos zonas (izquierda / derecha)
  • Horizontal: dos zonas (arriba / abajo)
  • Checkerboard: cuatro zonas en forma de tablero de ajedrez (2x2)
  • Vertical4Columns: cuatro zonas en columnas verticales

Cada layout implementa la lógica para calcular sus zonas en función de las dimensiones de la pantalla, maneja correctamente las divisiones impares para evitar huecos, y expone la cantidad de zonas que genera para validar la configuración de la terapia.
