pub fn init(app: &mut App) -> Result<(), Box<dyn Error>>
Expand description

Inicializa la aplicación Tauri.

Esta función se llama desde lib.rs en el setup del tauri::Builder. Configura todos los componentes necesarios para que la aplicación funcione.

§Argumentos

  • app - La aplicación Tauri en construcción.

§Errores

Devuelve un error si falla la creación de directorios, la carga de configuración o la creación de la bandeja.

§Ejemplo

use tauri::App;
use terapia_visual_app_lib::setup::init;

init(app)?;
